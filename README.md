# wp-plugin-MyFunctions
MyFunctions is a simple WordPress plugin that allows you to add custom PHP code directly into the `functions.php` file of the active theme through an easy-to-use admin interface.


## Folder Structure
```
myfunctions/
│
├── css/
│   └── style.css
├── js/
│   └── script.js
├── presets/
│   └── (.php files)
├── myfunctions.php
```
